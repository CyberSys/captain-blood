Спецификация на корабли для морских миссий.

1. корпус из 8-ми частей
2. иерархии частей не нужны
3. паруса не нужны
4. веревки (ванты и пр.) не нужны
5. именовать части корабля по принципу:

- части корпуса
по левому борту
hullL1, hullL2, hullL3, hullL4
по правому борту
hullR1, hullR2, hullR3, hullR4

- мачты
mastXY, где
X - номер мачты считая с носа корабля
Y - номер куска мачты считая снизу вверх

смотровые площадки на мачтах с куском мачты в единый объект

- реи
yardX1, yardX2 yardX3, где X - номер мачты на которой рея.
нумерация рей снизу вверх.
реи на куски разбивать не надо.
