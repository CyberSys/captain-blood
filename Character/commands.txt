Список команд, поддерживаемых персонажем.

---------------------------------------------------------------
ET_Log
---------------------------------------------------------------
Выводить в LogicDebug дополнительную информацию о EffectTable


---------------------------------------------------------------
EnableActor
---------------------------------------------------------------

Блокировать управляемость и колижен персонажа

  Параметры:
  c  EnableActor


---------------------------------------------------------------
DisableActor
---------------------------------------------------------------

Вернуть управляемость и колижен персонажа

  Параметры:
  c  DisableActor

---------------------------------------------------------------
Goto
---------------------------------------------------------------

Перейти на заданную точку в объекте Time line

  Параметры:
  c  Goto
  1  индекс итема в объекте Time line


---------------------------------------------------------------
ActorSet
---------------------------------------------------------------

Установить новый time line актёру

  Параметры:
  c  ActorSet
  1  "имя объекта time line"


---------------------------------------------------------------
AddItem
---------------------------------------------------------------

Добавить нужное количество предметов персонажу в инвентарь

  Параметры:
  c  AddItem
  1  "Идентификатор предмета"
  2  "Моделька предмета"
  3  "Локатор на котором размещать предмет"
  
  
  c  AddItem
  1  "Идентификатор предмета"
  2  "Моделька предмета"
  3  "Локатор на котором размещать предмет"  
  4  "trail" добавить возможность включать след


---------------------------------------------------------------
DelItem
---------------------------------------------------------------

Удалить предметы из инвентаря

  Параметры:
  c  DelItem
  1  "Идентификатор предмета"


---------------------------------------------------------------
SetHP
---------------------------------------------------------------

Установить количество жизни персонажу, если ставиться
больше максимального, то оно ограничеться максимальным.

  Параметры:
  c  SetHP
  1  "количество хп" (число с плавающей точкой)


---------------------------------------------------------------
ChangeHP
---------------------------------------------------------------

Изменить количество HP, для увеличения использовать
положительные числа, а для уменьшения отрицательные.

  Параметры:
  c  ChangeHP
  1  "на сколько изменить" (число с плавающей точкой)

  
---------------------------------------------------------------
GizmoRespawn
---------------------------------------------------------------
 
Отреспанить персонажа в указанной ремпавн гизмо
  c Respawn
  1  "имя респавн гизмы"

	
---------------------------------------------------------------
Respawn
---------------------------------------------------------------
 
Переродить персонажа
  c Respawn

Переродить персонажа с телепортом    
  c Respawn
  1  "имя миссионого объекта"


---------------------------------------------------------------
Show
---------------------------------------------------------------
 
Показать персонажа
  c Show

---------------------------------------------------------------
Hide
---------------------------------------------------------------
 
Скрыть персонажа
  c Hide

---------------------------------------------------------------
Activate
---------------------------------------------------------------
 
Активировать персонажа
  c Activate

---------------------------------------------------------------
Deactivate
---------------------------------------------------------------
 
Деактивировать персонажа
  c Deactivate

---------------------------------------------------------------
Instant kill
---------------------------------------------------------------

Мгновенно убить персонажа без плавного растворения
  c Instant kill


---------------------------------------------------------------
Teleport
---------------------------------------------------------------

Мгновенно телепортировать песона
  c Teleport
  1 "имя миссионого объекта"



---------------------------------------------------------------
EnableActorColider
---------------------------------------------------------------

Включить колижин персонажу в режиме актера
  c EnableActorColider  
	
	
---------------------------------------------------------------
DisableActorColider
---------------------------------------------------------------

Выключить колижин персонажу в режиме актера
  c EnableActorColider  


---------------------------------------------------------------
EnableAniBlend
---------------------------------------------------------------

Включить блендинг между актерским графом и графом персонажа
  c EnableAniBlend
	

---------------------------------------------------------------
DisableAniBlend
---------------------------------------------------------------

Выключить блендинг между актерским графом и графом персонажа
  c DisableAniBlend
	

---------------------------------------------------------------
Setwaypoint
---------------------------------------------------------------

Задать имя системы путей
  c setwaypoint
	
---------------------------------------------------------------
Settarget
---------------------------------------------------------------

Задать имя цели атаки персонажу, управляемому AI
  c settarget
  1 "имя миссионого объекта"
	

---------------------------------------------------------------
Setbombtarget
---------------------------------------------------------------

Задать цель для бросания бомбы 
  c setbombtarget
  1 "имя миссионого объекта"



---------------------------------------------------------------
movelocator
---------------------------------------------------------------

Переместить логический локатор
  с movelocator
  1 имя логичекого локатора  
  2 имя локатора куда переместить логический локатор
